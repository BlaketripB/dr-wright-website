## Deployment

Static site — deploy to any host (Vercel, Netlify, GitHub Pages, Hostinger).

```bash
# GitHub Pages
git init
git add .
git commit -m "initial commit"
git remote add origin <your-repo-url>
git push -u origin main
```

Then connect to Vercel or point your DNS from Hostinger.
